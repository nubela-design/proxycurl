Person Profile Endpoint

Input

Person Profile URL

Output

Profile data with profile picture, job history, etc. 

       
curl \
    -X GET \
    -H "Authorization: Bearer ${YOUR_API_KEY}" \
    https://nubela.co/proxycurl/api/v2/linkedin?url=https%3A%2F%2Fwww.linkedin.com%2Fin%2Fjoho-balboa%2F&use_cache=if-present

Company Profile Endpoint

Input

Company Profile URL

Output

Profile data with profile picture, office locations, etc. 


curl \
    -X GET \
    -H "Authorization: Bearer ${YOUR_API_KEY}" \
    https://nubela.co/proxycurl/api/linkedin/company?categories=include&funding_data=include&extra=include&exit_data=include&acquisitions=include&url=https%3A%2F%2Fwww.linkedin.com%2Fcompany%2Fapple%2F&use_cache=if-present
