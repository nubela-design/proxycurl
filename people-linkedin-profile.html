Request to scrape a Linkedin Profile

With one (slightly-long) line of code 

                

response = requests.get('https://nubela.co/proxycurl/api/linkedin',
              params={'url': 'https://www.linkedin.com/in/williamhgates'},
              headers={'Autorization': 'Bearer '+ API_KEY})
